Suckless irc (ii) in vim
########################

WIP
===

Thin layer for ``ii`` to do irc in vim.

``ii`` has to be installed.

.. code::

  :IIJoin irc.libera.chat #vim
  :IIJoin irc.libera.chat #perl
  :IIJoin irc.libera.chat #python
